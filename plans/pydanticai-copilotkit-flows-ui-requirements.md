# PydanticAI + CopilotKit: Flows → UI Requirements

## Goal
Turn the UI flows in `spec/ui/pydanticai_copilotkit_integration.md` into clear, implementable requirements:
- polished user-facing flow descriptions
- concrete UI components and interaction sequences
- happy paths + failure/edge cases (and how we notify users)
- long-running task handling (multiple tool calls, progress, retries)
- mapping to CopilotKit capabilities + PydanticAI (AG-UI) primitives
- a build order with frontend/backend deliverables per milestone

## Non-goals
- Implement the Next.js frontend or any backend routes in this task.
- Decide final visual design beyond component-level requirements.
